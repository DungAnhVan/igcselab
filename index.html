How it works

From weak point diagnosis to a practical study plan

Diagnostic test

A 30-45 minute session checks the student's current thinking, not only final answers.

Weak point map

We identify subject gaps, process errors, and exam-technique issues that repeat across questions.

Personal roadmap

Parents receive clear learning priorities and a suggested order for rebuilding foundations.

Practice plan

If tutoring is needed, sessions focus on targeted rebuilding and exam-style practice.

Parent trust

Calm, transparent support for international-school families

IGCSE Lab is built for parents who want a clear academic explanation before committing to long-term tutoring.

  • Clear diagnostic summary after the first session.
  • Parent-friendly language, with short Vietnamese clarification when useful.
  • Focus on core thinking, not memorising answer patterns.
  • No promise of perfect grades or one-size-fits-all packages.
